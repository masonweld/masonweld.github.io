Simulating the James-Stein Estimator

Let $X = (X_1, X_2, ..., X_p) \sim N_p(\vec{\theta}, I)$, where $\vec{\theta}$ is the vector of means which is unkown. We show via simulation that having been provided a single observation $\vec{x}$, the James-Stein (Shrinkage) Estimator has a lower squared-error of predicting the mean vector than the seemingly intuitive observation values $\vec{x}$ (i.e. the maximum likelihood estimator) when $p \geq 3$.

The below simulation randomly assigns the true $\vec{\theta}$ and simulates from $X$. For each trial, we assign the MLE as the sample values. We then apply the James-Stein Shrinkage factor $(1-\frac{p-2}{\|x\|_2^2})$ to the MLE to derive the new estimate. Lastly, the total squared-error is calculated for both estimates. This process is repeated for higher dimsensions and the distributions of squared-errors are plotted to visually show the improvement of the James-Stein estimator in higher dimensions.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

def run_simulation(dimensions, n_trials=10000):
    """
    Runs a Monte Carlo simulation to compare the MSE of MLE and James-Stein estimators.
    """
    results = []
    
    # Use a fixed true mean for reproducibility. A mean near the origin
    # highlights the shrinkage effect.
    np.random.seed(42)
    true_means = np.random.randn(len(dimensions)+1) * 0.5 

    for p in dimensions:
        theta = true_means[:p]
        
        # Generate n_trials observations from N(theta, I)
        # Each row is a trial, columns are dimensions
        x = np.random.randn(n_trials, p) + theta
        
        # --- Calculate Estimators ---
        
        # 1. Maximum Likelihood Estimator (MLE) is just the observation x
        mle_estimates = x
        
        # 2. James-Stein Estimator (positive-part)
        # Calculate the squared norm of each observation vector x
        squared_norm_x = np.sum(x**2, axis=1)
        
        # Calculate the shrinkage factor, ensuring it's not negative
        # Note: This is broadcasted across each vector in the trial
        shrinkage_factor = np.maximum(0, 1 - (p - 2) / squared_norm_x)
        
        # Apply shrinkage
        js_estimates = (shrinkage_factor[:, np.newaxis] * x)
        
        # --- Calculate Squared Errors ---
        mle_sq_error = np.sum((mle_estimates - theta)**2, axis=1)
        js_sq_error = np.sum((js_estimates - theta)**2, axis=1)
        
        # Store results for this dimension
        for i in range(n_trials):
            results.append([p, 'MLE', mle_sq_error[i]])
            results.append([p, 'James-Stein', js_sq_error[i]])
            
    return pd.DataFrame(results, columns=['Dimension', 'Estimator', 'Squared Error'])

# --- Main Execution ---

# Define the dimensions to test
dims_to_test = range(2, 11)
simulation_results = run_simulation(dims_to_test)

# --- Plotting ---

plt.style.use('seaborn-v0_8-whitegrid')
fig, ax = plt.subplots(figsize=(12, 8))

# Create the grouped boxplot
sns.boxplot(
    data=simulation_results,
    x='Dimension',
    y='Squared Error',
    hue='Estimator',
    palette={'MLE': 'skyblue', 'James-Stein': 'salmon'},
    ax=ax
)

# Calculate and display mean MSEs for clarity
mean_mses = simulation_results.groupby(['Dimension', 'Estimator'])['Squared Error'].mean().unstack()
print("Mean Squared Error (MSE) by Dimension:")
print(mean_mses)

# Improve plot aesthetics
ax.set_title('James-Stein vs. MLE: Mean Squared Error Comparison', fontsize=16, fontweight='bold')
