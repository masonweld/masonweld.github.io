Maximum Likelihood Estimation for Mixture Models¶

Finding the maximum likelihood can prove to be more difficult for models with more than one component. As described in the section, below is an implementation of an Expectation Maximization algorithm to model maximum likelihood estimation given an example of a model with two components. Before discussion of the model and algorithm, first we import the necessary Python libraries.

As described in the book, a reasonable observation of the distribution of the data, given its two peaks, is that it can be modeled as the combination of two normal distributions (8.36):

$$ Y \sim (1-\Delta) N(\mu_1, \sigma_1^2) + \Delta N(\mu_2, \sigma_2^2)$$

Where $\Delta$ is either zero or one. Thus, and observation of $Y$ either comes from the first or second normal distribution based on the value of $\Delta$. The probability that $\Delta_i = 1$ is denoted as $\pi$. Our goal is to find the parameters $\bar{\mu}, \bar{\sigma^2}, \pi$ that maximize the log-likelihood function of $Y$. As described in the book, finding the maximum likelihood estimates for $\bar{\mu}$ and $\bar{\sigma}$ is easy for each individual component. Thus, we can use the fact that $\Delta \in \{0,1\}$ to set a guess of the frequency $\Delta_i = 1$ as $\gamma_i = $ probablity $\Delta_i = 1$ given our estimats of $\bar{\mu_i}, \bar{\sigma_i^2}$ and $\pi$. The idea will then be to iteratively maximize the log-likelihood by updating each parameter.

The psuedocode of the algorithm is on page 275.

The first step is to set reasonable initial estimates for the parameters we are estimating. The text suggests the sample variance for both variance estiamtes, two random data points for the mean estimates, and for mixing probability $\pi = \frac{1}{2}$.

In [3]:
var = sum((data - data.mean()) ** 2) / N * np.array([1, 1])
mu = data[np.random.choice(N, 2, replace=False)]
pi = 0.5

The next step is implementing the iterative optimization process of the algorithm. This consists of first computing the expectation of each $\gamma_i$ as described above, and using the updated $\gamma$ to recalculate the sample means and variances weighted by the expectation of $\Delta$ modeled from $\gamma$ in the maximization step. Once the stopping condition is met, the parameters are kept as our maximum likelihood estimates.

In [4]:
cnt = 1
while 1:
    print("Iteration " + str(cnt) + ": pi ~= " + str(round(pi,3)))
    gamma = np.zeros(N)
    
    #Expectation Step
    for i in range(N):
        gamma[i] = ( (pi * norm.pdf(data[i], mu[1], np.sqrt(var[1]))) / 
                     ((1 - pi) * norm.pdf(data[i], mu[0], np.sqrt(var[0])) + pi * norm.pdf(data[i], mu[1], np.sqrt(var[1]))) )
    
    #Maximization Step
    mu[0] = sum((1-gamma)*data) / sum(1-gamma)
    mu[1] = sum(gamma*data) / sum(gamma)
    var[0] = sum((1-gamma)*(data-mu[0])**2) / sum(1-gamma)
    var[1] = sum(gamma*(data-mu[1])**2) / sum(gamma)
    lastpi = pi
    pi = sum(gamma) / N
    
    #Stopping Condition
    if abs(pi - lastpi) < 10e-4:
        break
    cnt += 1

print(("mu_1 = " + str(round(mu[0],3)) + ", mu_2 = " + str(round(mu[1],3)) + "\n" + 
       "var_1 = " + str(round(var[0],3)) + ", var_2 = " + str(round(var[1],3))))

To visualize how our parameters fit the underlying data and bi-modal distribution assumption, we can use the derived probability density function to plot it against the distribution of the given data. First, the pdf is simply given by:

$$g_Y(x) = (1-\Delta) f(x, \mu_1, \sigma_1) + \Delta f(x, \mu_2, \sigma_2)$$

Where $f$ is the normal pdf, given as $f(x, \pi, \sigma) = \frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{1}{2}(\frac{x-\mu}{\sigma})^2}$.

In [5]:
def g(x, pi, mu, var):
    return (1-pi)*norm.pdf(x, mu[0], np.sqrt(var[0])) + (pi)*norm.pdf(x, mu[1], np.sqrt(var[1]))

Now we can plot our pdf versus the given data, and we see that our estimates of the parameters provided a good estimate of the data given. With our derived pdf, we can be more confident in the underlying distribution of sampled data.

Generalizing the EM Algorithm for M Guassian Components¶

In order to generalize the above algorithm for $M$ normal components, first we need to generalize the assumption of the underlying distribution. \begin{align} &\sum_i^M \Delta_i = 1, \; \Delta_i \in \{0,1\} \; \forall i \\ &P(\Delta_i = 1) = \pi_i, \; \forall i \\ &Y \sim \sum_i^M \Delta_i N(\mu_i, \sigma_i^2) \end{align} The same initial guesses will be used, with the change of $\pi_i = \frac{1}{M}$ for all $i$ to ensure $\sum_{i=1}^M \pi_i = 1$. The function below takes the data as an input, along with the number of mixtures we are assuming $M$.

In [7]:
def Expectation_Maximization(data, M):
    
    N = len(data)
    
    var = sum((data - data.mean()) ** 2) / N * np.array([1 for i in range(M)])
    mu = data[np.random.choice(N, M, replace=False)]
    pi = np.array([1/M for i in range(M)])

    cnt = 1
    while 1:
        if cnt % 10 == 0 or cnt == 1:
            print("Iteration " + str(cnt) + ": pi ~= ")
            print(pi)
        
        #Expectation Step
        gamma = np.zeros((N,M))
        for i in range(N):
            for j in range(M):
                gamma[i,j] = ( pi[j]*norm.pdf(data[i], mu[j], np.sqrt(var[j])) / 
                             sum(pi*norm.pdf(data[i], mu, np.sqrt(var))) )
        
        #Maximization Step
        lastpi = pi
        for j in range(M):
            mu[j] = sum(gamma[:,j]*data) / sum(gamma[:,j])
            var[j] = sum(gamma[:,j]*(data-mu[j])**2) / sum(gamma[:,j])
            pi[j] = sum(gamma[:,j]) / N

        if cnt > 50:
            break
        cnt += 1
    
    return (mu, var, pi)

We also need an updated function that calculates the density function given parameters of any length.

In [8]:
def g_general(x, pi, mu, var):
    return sum(pi*norm.pdf(x, mu, np.sqrt(var)))

Now we can generate some data split from $M$ normal distributions with random parameters. We can then run the algorithm, and see how well the estimation fits the underlying data. The below generates data for $M =$ 3, 4, and 5, runs the model, and plots the results.
